# FAPI-KR3
Контрольная по дисциплине "Технологии разработки серверных приложений" №3

## 📖 Что здесь освещается

Проект демонстрирует полный цикл разработки современного REST API приложения:

**Аутентификация & Безопасность:**
- JWT токены и OAuth2 потоки
- Хеширование паролей с bcrypt
- Role-Based Access Control (RBAC)
- Защита от SQL Injection

**Backend разработка:**
- FastAPI для создания API
- Pydantic для валидации данных
- SQLite для хранения данных
- Асинхронное программирование (async/await)
